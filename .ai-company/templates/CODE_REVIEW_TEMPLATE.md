## Review

- [ ] 变更与批准范围一致。

- [ ] 类型、错误和失败路径合理。
- [ ] 权限、租户和敏感数据安全。
- [ ] 无无关重构或隐藏行为变化。
- [ ] 测试覆盖关键风险。
- [ ] 数据迁移和兼容方案完整。
- [ ] 性能和并发影响可接受。
- [ ] 日志、监控和运行影响已处理。
- [ ] 文档和变更说明已更新。
